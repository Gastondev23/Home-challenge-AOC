Gallery Project Execution Guide

This document provides step-by-step instructions on how to execute and run the Gallery Project on your local machine.

Prerequisites

Ensure you have the following software installed on your machine: A web browser (e.g., Google Chrome, Mozilla Firefox, Safari) Git (for cloning the repository)

Steps to Execute

1. Clone the Repository

First, you need to clone the repository from GitHub. Open your terminal or command prompt and run the following command:
  • git clone https://github.com/benjamin-mku/TAKE-HOME-CHALLENGES.git

    • 2. Navigate to the Project Directory

    Change to the project directory using the following command:
  • cd TAKE-HOME-CHALLENGES
  • Choose Gallery one or Two

    • 3. Open the Project in a Web Browser

    Open the index.html file in your preferred web browser. You can do this in several ways: Method 1: Using the Command Line (if supported)
  • open index.html # For macOS
  • start index.html # For Windows
  • xdg-open index.html # For Linux
    • Method 2: Manually Open your web browser. Use the "File" menu to open a file, then navigate to the project directory and select 'index.html'.

    4. Explore the Gallery

    Once index.html is opened in your web browser, you will see the image gallery. You can:
  • hover your mouse to images to see changes
  • Enjoy smooth transitions and animations.

    • Troubleshooting

    Common Issues

    Images not loading: Ensure all images are placed in the images folder and the paths are correctly referenced in the index.html file.
    CSS not applied: Check the path to your style.css file in the index.html file to ensure it is correct.
    JavaScript not functioning: Open the browser's developer tools (usually by pressing F12 or Ctrl+Shift+I) and check the console for any errors.

    Tips

    Make sure your file paths are relative and correctly pointing to the respective folders and files. If changes are not reflecting, try clearing your browser cache or performing a hard reload (usually Ctrl+Shift+R).
